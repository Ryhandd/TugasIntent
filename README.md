# Tugas Pemrograman Mobile: Implementasi Intent

Proyek ini mendemonstrasikan penggunaan **Intent** di Android Studio untuk melakukan navigasi antar halaman (Activity). 

## 📋 Deskripsi Tugas
Aplikasi ini terdiri dari dua halaman utama:
1. **MainActivity (Halaman 1):** Berisi sebuah tombol (Button) untuk memicu perpindahan halaman.
2. **SecondActivity (Halaman 2):** Halaman tujuan yang ditampilkan setelah tombol di halaman pertama diklik.

## ⚙️ Cara Kerja
* Menggunakan kelas `Intent` untuk mendefinisikan asal dan tujuan Activity.
* Menggunakan method `startActivity()` untuk mengeksekusi perpindahan.
* Implementasi `setOnClickListener` pada komponen Button di halaman utama.

## 🛠️ Tech Stack
* **IDE:** Android Studio
* **Bahasa:** [Java / Kotlin]
* **Komponen:** Intent, Activity Lifecycle, XML Layout.

## 📂 Struktur File Penting
* `MainActivity.java/kt` -> Logika pemicu Intent.
* `SecondActivity.java/kt` -> Activity tujuan.
* `activity_main.xml` -> Layout dengan tombol navigasi.
* `activity_second.xml` -> Layout halaman kedua.
